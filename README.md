# 이예림 포트폴리오

> 소프트웨어 × 금융공학을 전공하며 데이터 · 블록체인 · AI로 금융을 탐구합니다.

[![Live Demo](https://img.shields.io/badge/Live-Demo-3182f6?style=flat-square)](https://yelim8902.github.io/portfolio-website/)
[![GitHub](https://img.shields.io/badge/GitHub-yelim8902-181717?style=flat-square&logo=github)](https://github.com/yelim8902)

## 소개

아주대학교 소프트웨어학과 / 금융공학과 복수전공 재학생의 개인 포트폴리오 웹사이트입니다.
블록체인 보안, 데이터 분석, AI를 금융 도메인에 접목하는 프로젝트들을 소개합니다.

## 주요 기능

- **다크모드 토글** — 수동 전환 및 localStorage 저장
- **스크롤 진행률 바** — 상단 고정 진행 표시
- **타이핑 애니메이션** — Hero 타이틀 글자 등장 효과
- **Intersection Observer** — 스크롤 기반 섹션 등장 애니메이션
- **반응형 디자인** — 모바일 / 태블릿 / 데스크톱 지원
- **EmailJS** — Contact 폼 이메일 전송

## 기술 스택

| 분류 | 기술 |
|---|---|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Custom Properties, Grid, Flexbox) |
| 스크립트 | Vanilla JavaScript (ES6+) |
| 폰트 | Pretendard Variable |
| 아이콘 | Font Awesome 6 |
| 이메일 | EmailJS |
| 배포 | GitHub Pages |

3. **Projects** — 주요 프로젝트 (블록체인 AML, XAI, 배당주 시뮬레이션 등)
4. **Skills** — Programming / DB & Tools / Web / Design
5. **Awards** — 수상 경력 (이미지 포함)
6. **Activities** — 활동 내역 타임라인
